# Funffy Gut Check

免费的犬猫肠胃健康筛查静态网页，可从 WhatsApp 链接直接打开。

## 本地预览

在此目录运行：

```powershell
python -m http.server 8000
```

然后打开 `http://localhost:8000`。

## 零费用部署

推荐使用 GitHub Pages：

1. 新建一个公开 GitHub repository。
2. 上传本目录全部文件。
3. 在 repository 的 `Settings → Pages` 中选择从 `main` 分支部署。
4. 等待生成 `https://你的用户名.github.io/项目名/` 链接。
5. 将链接发送到 WhatsApp。

GitHub Pages 适合当前版本，因为它没有服务器、数据库或 API 费用。

## 数据与隐私

- 测试无需注册。
- 回答不会上传服务器。
- 最近一次结果只保存在顾客当前设备的浏览器中。
- 清除浏览器资料后，结果会被删除。

## 上线前

当前评分逻辑用于产品原型和健康教育，不是兽医诊断。正式公开推广前，应由执业兽医审核题目、阈值、紧急警示和报告措辞。
